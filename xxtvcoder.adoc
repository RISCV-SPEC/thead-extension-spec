== Vector Coder Turbo instructions (XXTVcoder)

[NOTE,caption=Frozen]
The `XXTVcoder` extension is `stable`.

The `XXTVcoder` ISA extension provides vector coder turbo instructions including absolute difference, widening absolute difference, absolute difference accumulate, floating-point absolute difference, and absolute max operations.

This extension depends on the availability of the `V` (vector) ISA extension.

Extension version: 1.0.

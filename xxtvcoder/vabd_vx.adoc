[#xxtvcoder-insns-vabd-vx,reftext=Vector absolute value of the difference (vector-scalar)]
==== xt.vabd.vx

Synopsis::
Vector absolute value of the difference.

Mnemonic::
xt.vabd.vx _vd_, _vs2_, _rs1_, _vm_

Encoding::
[wavedrom, , svg]
....
{reg:[
    { bits:  7, name: 0xb, attr: ['custom-0, 32 bit'] },
    { bits:  5, name: 'vd' },
    { bits:  3, name: 0x6 },
    { bits:  5, name: 'rs1' },
    { bits:  5, name: 'vs2' },
    { bits:  1, name: 'vm' },
    { bits:  6, name: '0x33' },
]}
....

Description::
Subtract each corresponding element of the source vector and the source scalar and then return the absolute value to the destination vector. The operands are signed numbers.

Operation::
[source,sail]
--
vd[i] = abs(vs2[i] - rs1)
--

Permission::
This instruction can be executed in all privilege levels.

Exceptions::
This instruction does not trigger any exceptions.

[#xxtvcoder-insns-vwabdu-vv,reftext=Vector widening unsigned absolute difference (vector-vector)]
==== xt.vwabdu.vv

Synopsis::
Vector absolute value of the difference obtained by zero-extended unsigned numbers.

Mnemonic::
xt.vwabdu.vv _vd_, _vs2_, _vs1_, _vm_

Encoding::
[wavedrom, , svg]
....
{reg:[
    { bits:  7, name: 0xb, attr: ['custom-0, 32 bit'] },
    { bits:  5, name: 'vd' },
    { bits:  3, name: 0x7 },
    { bits:  5, name: 'vs1' },
    { bits:  5, name: 'vs2' },
    { bits:  1, name: 'vm' },
    { bits:  6, name: '0x3a' },
]}
....

Description::
After the unsigned expansion of each element of the two source registers, the corresponding elements are subtracted and then returned to the destination register as absolute values.

Reserved Encodings::
SEW is any value other than 8, 16, 32.

Operation::
[source,sail]
--
vd[i] = abs(unsigned(vs2[i]) - unsigned(vs1[i]))
--

Permission::
This instruction can be executed in all privilege levels.

Exceptions::
This instruction does not trigger any exceptions.

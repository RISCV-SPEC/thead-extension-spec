[#xxtvcoder-insns-vfabd-vv,reftext=Vector floating-point absolute difference (vector-vector)]
==== xt.vfabd.vv

Synopsis::
Vector floating-point absolute value of the difference.

Mnemonic::
xt.vfabd.vv _vd_, _vs2_, _vs1_, _vm_

Encoding::
[wavedrom, , svg]
....
{reg:[
    { bits:  7, name: 0xb, attr: ['custom-0, 32 bit'] },
    { bits:  5, name: 'vd' },
    { bits:  3, name: 0x7 },
    { bits:  5, name: 'vs1' },
    { bits:  5, name: 'vs2' },
    { bits:  1, name: 'vm' },
    { bits:  6, name: '0x34' },
]}
....

Description::
Subtract each floating-point element of the two source registers and then return the absolute value to the destination registers.

Operation::
[source,sail]
--
vd[i] = abs(vs2[i] - vs1[i])
--

Permission::
This instruction can be executed in all privilege levels.

Exceptions::
This instruction does not trigger any exceptions.

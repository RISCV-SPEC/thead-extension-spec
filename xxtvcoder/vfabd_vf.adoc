[#xxtvcoder-insns-vfabd-vf,reftext=Vector floating-point absolute difference (vector-scalar)]
==== xt.vfabd.vf

Synopsis::
Vector floating-point absolute value of the difference.

Mnemonic::
xt.vfabd.vf _vd_, _vs2_, _rs1_, _vm_

Encoding::
[wavedrom, , svg]
....
{reg:[
    { bits:  7, name: 0xb, attr: ['custom-0, 32 bit'] },
    { bits:  5, name: 'vd' },
    { bits:  3, name: 0x7 },
    { bits:  5, name: 'rs1' },
    { bits:  5, name: 'vs2' },
    { bits:  1, name: 'vm' },
    { bits:  6, name: '0x35' },
]}
....

Description::
Subtract each floating-point element of the source vector and the source scalar and then return the absolute value to the destination vector.

Operation::
[source,sail]
--
vd[i] = abs(vs2[i] - f[rs1])
--

Permission::
This instruction can be executed in all privilege levels.

Exceptions::
This instruction does not trigger any exceptions.

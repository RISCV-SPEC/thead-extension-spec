[#xxtvcoder-insns-vaba-vv,reftext=Vector absolute difference and accumulate (vector-vector)]
==== xt.vaba.vv

Synopsis::
Vector absolute value of the difference and accumulate.

Mnemonic::
xt.vaba.vv _vd_, _vs2_, _vs1_, _vm_

Encoding::
[wavedrom, , svg]
....
{reg:[
    { bits:  7, name: 0xb, attr: ['custom-0, 32 bit'] },
    { bits:  5, name: 'vd' },
    { bits:  3, name: 0x6 },
    { bits:  5, name: 'vs1' },
    { bits:  5, name: 'vs2' },
    { bits:  1, name: 'vm' },
    { bits:  6, name: '0x36' },
]}
....

Description::
Subtract each corresponding element of the two source registers and then compute the absolute value, accumulating to the destination registers. The operands are signed numbers.

Operation::
[source,sail]
--
vd[i] += abs(vs2[i] - vs1[i])
--

Permission::
This instruction can be executed in all privilege levels.

Exceptions::
This instruction does not trigger any exceptions.

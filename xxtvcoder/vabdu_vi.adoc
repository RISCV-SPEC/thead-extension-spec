[#xxtvcoder-insns-vabdu-vi,reftext=Vector unsigned absolute value of the difference (vector-immediate)]
==== xt.vabdu.vi

Synopsis::
Vector unsigned absolute value of the difference.

Mnemonic::
xt.vabdu.vi _vd_, _vs2_, _uimm_, _vm_

Encoding::
[wavedrom, , svg]
....
{reg:[
    { bits:  7, name: 0xb, attr: ['custom-0, 32 bit'] },
    { bits:  5, name: 'vd' },
    { bits:  3, name: 0x6 },
    { bits:  5, name: 'uimm' },
    { bits:  5, name: 'vs2' },
    { bits:  1, name: 'vm' },
    { bits:  6, name: '0x27' },
]}
....

Description::
Subtract each corresponding element of the source vector and the source immediate and then return the absolute value to the destination vector. The operands are unsigned numbers.

Operation::
[source,sail]
--
vd[i] = abs(vs2[i] - uimm)
--

Permission::
This instruction can be executed in all privilege levels.

Exceptions::
This instruction does not trigger any exceptions.
